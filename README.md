# Trae Chat

一个基于Trae AI的聊天应用项目。

## 功能特性

- 实时聊天功能
- 支持多种消息类型
- 智能AI助手集成
- 现代化用户界面

## 技术栈

- React/Next.js
- Node.js
- Trae AI API
- WebSocket

## 安装和运行

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```
